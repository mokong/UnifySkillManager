Generate app icons for iOS/macOS/Safari Extension projects from a source image. # App Icon Generator

Generate all required app icons from a single source image for iOS, macOS, and Safari Extension projects.

## Quick Start

1. Verify source image exists (supports PNG, JPG, JPEG, HEIC, HEIF)
2. Check source image dimensions (1024x1024 recommended)
3. Generate all required icon sizes
4. Update Contents.json if needed

## Commands

Use `sips` (built-in macOS tool) to resize images:

```bash
# Basic resize to PNG format
sips -s format png <source> --out <output> --resampleWidth <width>

# Example: Generate 128x128 icon
sips -s format png ~/Downloads/image.jpg --out ./icon-128.png --resampleWidth 128
```

### Generate SVG with Embedded PNG

For toolbar-icon.svg, create an SVG with base64-encoded PNG:

```bash
# 1. Generate small PNG
sips -s format png <source> --out /tmp/icon-temp.png --resampleWidth 32

# 2. Get base64 string
base64 -i /tmp/icon-temp.png

# 3. Create SVG file with embedded image
```

SVG template:
```xml
<svg xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" width="32" height="32" viewBox="0 0 32 32">
  <image width="32" height="32" xlink:href="data:image/png;base64,<BASE64_STRING>"/>
</svg>
```

## Workflow

1. **Locate source image** - Check user-provided path, fallback to common locations
2. **Verify format** - Use `sips -g all <image>` to check dimensions and format
3. **Identify target directories**:
   - `Shared (App)/Assets.xcassets/AppIcon.appiconset/`
   - `Shared (App)/Resources/`
   - `Shared (Extension)/Resources/images/`
4. **Generate all icons** - Run sips commands for each required size
5. **Update configuration** - Update Contents.json with new filenames
6. **Verify** - List generated files to confirm success
